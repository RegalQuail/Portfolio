Skeletet

Det vigtigste kode er <meta charset="UTF-8">. Den lader mig skrive ord med 'æ', 'ø' og 'å'. Hele min kode er kasset ind i <section class="tab">, med 4 hoved dele indlejret i hver deres <div class="tab-body">. Den første har "active" til sidste, fordi det er den, som er active først når man ankommer på siden. Hver hoved dele er delt i 2 med "tab-col-left" og "tab-col-right", hvor den venstre side har billeder, som indsættes således: <img src="navn" alt="tekst">, mens højre siden har tekst, og en knap under den. I kan læse mere om HTML på W3Schools med linket nedenunder, eller gå mere i dybden i min kode på hovedsiden.

Udsigten

Det er mest med CSS at vi lader "the magic happen". Den er skrevet på dens egen fil med navn: navn.css. Den linker man til HTML filen med <link rel="stylesheet" href="navn.css"> mellem <head> elementet. Den er også delt i små grupper, for eksempel for min "navigation bar" på toppen, eller mine knapper ved bunden af min side. Det der springer mest frem er organisationen, og hvordan det går mere og mere i dybden for hver element af siden. I kan gå tilbage til hovedsiden for at tage en "closer look" i min kode, eller klik nedenunder til at lære mere om CSS på W3Schools.

W3Schools CSS

Musklerne

JavaScript kan være ret skræmmende når man starter med at se alt det den kan gør, men her er koden ret simpelt når man forstår den. JavaScript kan også skrives på dens egen fil med navn: navn.js, som linkes til HTML filen med en "script" tag således: <script src="navn.js"></script>. Det er koden der gøre at i kan skifte fra en side til en anden, ved at klikke på min "navigation bar". I kan lære mere om JavaScript på W3Schools nedenunder.
